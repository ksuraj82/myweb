ArrayList in Java

In Java, ArrayList is a dynamic array implementation of the List interface provided by the Java Collections Framework. It offers resizable arrays,
allowing for the creation of lists that can dynamically grow or shrink as needed.
ArrayList provides indexed access to its elements, enabling fast random access and iteration over its contents. Elements can be added, removed, or accessed by their index position. Below is a basic example demonstrating the usage of ArrayList in Java: 

        
            import java.util.ArrayList;

            public class ArrayListExample {
                public static void main(String[] args) {
                    // Create an ArrayList to store strings
                    ArrayList<String> list = new ArrayList<>();

                    // Add elements to the ArrayList
                    list.add("Apple");
                    list.add("Banana");
                    list.add("Orange");

                    // Access elements by index
                    System.out.println("First element: " + list.get(0));
                    System.out.println("Second element: " + list.get(1));

                    // Iterate over the elements
                    System.out.println("Elements:");
                    for (String fruit : list) {
                        System.out.println(fruit);
                    }

                    // Remove an element
                    list.remove("Banana");

                    // Check if an element exists
                    System.out.println("Contains Orange: " + list.contains("Orange"));

                    // Get the size of the ArrayList
                    System.out.println("Size of the ArrayList: " + list.size());
                }
            }
